# Project Modules Overview

Welcome to the hackthone-q-4 project modules. This section contains all the different components that make up our unified book project using Claude Code and Spec-Kit Plus.

## Available Modules

- [.specify](./specify) - Spec-Kit Plus configuration and memory
- [specs](./specs) - Specifications and requirements documentation
- [backend](./backend) - Backend API and services implementation
- [frontend](./frontend) - Docusaurus frontend and documentation
- [history](./history) - Project history and prompt records
- [AI Configs](./ai-configs) - Claude, Gemini, and Qwen configurations
- [Utilities](./utils) - Testing and verification utilities

Each module contains specific information, documentation, and resources related to that component of the project. Use the navigation to explore each module in detail.
