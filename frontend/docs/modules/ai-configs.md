# AI Configuration Modules

## Overview
The AI configuration modules (`.claude`, `.gemini`, `.qwen`) contain AI-specific configurations, instructions, and guidelines for different AI systems used in the project.

## Contents
- `.claude/` - Claude-specific configurations and instructions
- `.gemini/` - Gemini-specific configurations and instructions
- `.qwen/` - Qwen-specific configurations and instructions (including this file)

## Key Elements
- System prompts and instructions
- AI role definitions
- Context and knowledge specifications
- Output format guidelines
- Safety and constraint configurations

## Purpose
These modules ensure consistent AI behavior across different systems and provide standardized interfaces for AI-assisted development and content generation.
