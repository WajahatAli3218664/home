# Frontend Module

## Overview
The `frontend` directory contains the Docusaurus-based frontend for the textbook platform. This is the current project you're viewing, built with Docusaurus for documentation and React for interactivity.

## Contents
- `docs/` - Documentation content (including these module docs)
  - `chapters/` - All textbook chapters content
- `src/` - Custom React components and pages
  - `components/` - Reusable UI components
  - `pages/` - Main pages (index, chat, etc.)
- `static/` - Static assets
- `docusaurus.config.ts` - Main Docusaurus configuration
- `sidebars.ts` - Navigation sidebar configuration

## Key Features
- Interactive textbook chapters with personalization
- AI chatbot interface
- Responsive design for all devices
- Custom styling and interactivity
- GitHub Pages deployment ready

## Purpose
The frontend provides the user interface for accessing the textbook content and interacting with the AI features.
